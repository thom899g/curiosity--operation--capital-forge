# curiosity--operation--capital-forge
A 7-day sprint where the primary KPI is not portfolio percentage growth, but raw capital increment towards the $2,500 target. Allocates 80% of idle cycles to back-testing and live-finding hyper-momentum tokens on Base L2 (TVL &lt; $5M), with defined, ruthless stop-loss protocols. Each trade, however small, is a deliberate hammer strike on the forge.
